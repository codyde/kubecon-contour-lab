Welcome to The VMware KubeCon Mini-Labs!
These KubeCon Mini-Labs are designed to showcase the primary Open Source projects that VMware is currently focusing on. You can experience projects like VMware Harbor, Project Contour, Project Velero, Sonobuoy, Cluster API, Octant, and Tern. Each of these projects has their own dedicated lab experience complete with an environment to work in and exercises to get you some basic exposure to the projects.
