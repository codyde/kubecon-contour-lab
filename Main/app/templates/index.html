Welcome to the Contour Mini-Lab
This mini-lab is intended to demonstrate the capabilities of Project Contour. We will be using this application to create an number of HTTPProxy objects and define routes to stitch our services together. This application was created using Python, Flask, and a whole lot of love. You can find the source for all of these projects on GitHub in this repository.
